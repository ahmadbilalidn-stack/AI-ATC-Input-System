# AI-ATC-Input-System
Introducing a new ATC with Input System

## Instalation
1. Install a userscript manager such as:<br/>
   - Tampermonkey<br/>
   - Greasemonkey<br/>
2. Go to the script page or download the `ATC.js` file<br/>
3. Click "New Script" in your manager and paste the contents of the script<br/>
4. Save the script (make sure you enable the script)<br/>
5. Open GeoFS and click the headphone logo beside the watch recording flight<br/>
6. Login Puter AI<br/>
<br/>

## Features<br/>
- ATC AI Puter<br/>
- Request takeoff & landing<br/>
- METAR Reader, You can ask the weather in your location (beta)<br/>
- Input System, more accurate language<br/>

## Tutorial
1. Press Radio icon
2. Enter airport ICAO
3. Press Headphone icon
4. Talk to the ATC


## Not Working?<br/>
Make sure:<br/>

- You login Puter AI<br/>
- You're  using Tampermonkey (recomended)<br/>
